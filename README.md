## 🚀 Overview

This repository contains a powerful **AI Skill/Agent Definition** designed to act as a Senior Academic Research Assistant. It bridges the gap between raw research data and polished academic writing by enforcing rigorous methodological standards and precise formatting rules.

### ✨ Key Capabilities

| Feature | Description |
| :--- | :--- |
| **📚 APA 7th Mastery** | Automated formatting for citations, references, headings, tables, and statistical reporting. |
| **🧠 Theory Integration** | Ensures theoretical frameworks are woven throughout the article, not just isolated in one section. |
| **📊 Data Interpretation** | Converts raw statistical outputs (SPSS, R, Python) into narrative results with effect sizes and CIs. |
| **🛡️ Integrity Guard** | Built-in checks to prevent hallucinated citations, fabricated data, or unsupported claims. |
| **🏗️ Structured Workflow** | Guides users through IMRAD+ structure (Introduction, Methods, Results, And Discussion). |
